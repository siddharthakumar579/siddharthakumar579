### 🔹 [Vectorless_Pdf_RAG](https://github.com/siddharthakumar579/Vectorless_Pdf_RAG)
A lightweight RAG (Retrieval-Augmented Generation) system that answers questions from PDF files without using any vector database or embeddings. Uses BM25 keyword retrieval to find relevant chunks and Gemini to generate answers.

<img src="https://github-readme-stats.vercel.app/api/pin/?username=siddharthakumar579&repo=Vectorless_Pdf_RAG&theme=nord&border_color=7dd3fc&title_color=7dd3fc&text_color=c9d1d9&bg_color=0d1117" alt="Vectorless_Pdf_RAG pin"/>

| Layer | Technology |
|---|---|
| Retrieval | BM25 Keyword Retrieval (Vector-less RAG) |
| Language Model | Gemini API |
| Core Language | JavaScript |
| Input Handling | PDF Parsing |

**🔗 Code:** [github.com/siddharthakumar579/Vectorless_Pdf_RAG](https://github.com/siddharthakumar579/Vectorless_Pdf_RAG)
